Temperature Converter

Instructions

  • In the 'Value' section, enter the temperature value you want to convert.
  • After 'Value' select box, choose the temperature's unit.
  • In the 'Convert In' select box, choose the unit of temperature you want.
  • For results, select "Calculate" and if you want to restore the values to their default settings, select "Reset."
  • The output field will show the converted temperature value.
